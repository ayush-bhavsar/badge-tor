# Custom Badge Creator

A simple web application to create custom badges for your GitHub READMEs or any other markdown files.

## How to Use

1.  Open `index.html` in your web browser.
2.  Enter the desired `label` and `message` for your badge.
3.  Select a `color` using the color picker.
4.  Choose a `style` from the dropdown menu.
5.  The badge preview will update automatically.
6.  Copy the generated markdown link and paste it into your file.

## Features

-   Live preview of the badge.
-   Customizable label, message, color, and style.
-   Generates markdown link for easy embedding.
-   Simple and clean user interface.
-   Copy to clipboard functionality.
